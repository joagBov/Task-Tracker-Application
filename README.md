# Task Tracker Application - Project Report

## Executive Summary

The Task Tracker Application is a web-based task management system developed as a collaborative group project using C# backend and HTML/CSS frontend technologies. This application provides users with the ability to create, manage, and track tasks efficiently through an intuitive web interface.

## Core Features

The Task Tracker Application provides the following functionality:

1. **Task Management** - Create, read, update, and delete tasks
2. **Task Persistence** - Data storage and retrieval capabilities
3. **User Interface** - Responsive web-based interface for task interaction
4. **Task Properties** - Comprehensive task item attributes and metadata
